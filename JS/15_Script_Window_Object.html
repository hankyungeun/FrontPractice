Window 객체

Window 객체는 자바스크립트의 최상위 객체
크게 BOM과 DOM으로 나누어 짐


- BOM (Browser Object Model) : location 객체, screen 객체, history 객체

- DOM (Document Object Model) : document 객체

[Window.]open("url","창이름","창의 특징")

[window.]setTimeout(함수, 시간ms)

설정한 시간 후에 함수의 내용을 실행("한번만")

[window.]setInterval(함수,시간ms)

설정한 시간마다 함수 내용을 실행
